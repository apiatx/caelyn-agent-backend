### Problem 2: Add USDT.d and Others.d (dominance data)

USDT dominance and "Others" dominance are critical crypto sentiment indicators:
- Rising USDT.d = risk-off, money flowing to stables (bearish for alts)
- Falling USDT.d = risk-on, money flowing into crypto (bullish for alts)
- Rising Others.d = alt season signal
- Falling Others.d = BTC season / flight to quality

This data is available from CoinGecko's global market data (`global_market` / `cg_global`).

### Fix 2: Extract dominance data in compression

In the `_compress_crypto_scanner` function, extract dominance data from the CoinGecko global market data:
```python
    # Global market + dominance data
    cg_global = data.get("cg_global", {})
    cmc_global = data.get("cmc_global", {})
    
    # Extract dominance percentages
    market_cap_percentage = cg_global.get("market_cap_percentage", {})
    
    compressed["market_summary"] = {
        "total_market_cap": cg_global.get("total_market_cap", {}).get("usd") if isinstance(cg_global.get("total_market_cap"), dict) else cg_global.get("total_market_cap"),
        "total_volume_24h": cg_global.get("total_volume", {}).get("usd") if isinstance(cg_global.get("total_volume"), dict) else cg_global.get("total_volume"),
        "market_cap_change_24h": cg_global.get("market_cap_change_percentage_24h_usd"),
        "btc_dominance": market_cap_percentage.get("btc") or cg_global.get("btc_dominance") or cmc_global.get("btc_dominance"),
        "eth_dominance": market_cap_percentage.get("eth") or cg_global.get("eth_dominance") or cmc_global.get("eth_dominance"),
        "usdt_dominance": market_cap_percentage.get("usdt"),
        "others_dominance": None,  # Calculated below
    }
    
    # Calculate Others dominance = 100% - BTC - ETH - USDT - other top coins
    btc_dom = compressed["market_summary"].get("btc_dominance") or 0
    eth_dom = compressed["market_summary"].get("eth_dominance") or 0
    usdt_dom = compressed["market_summary"].get("usdt_dominance") or 0
    
    # Others.d = everything except BTC, ETH, USDT (rough proxy for altcoin activity)
    if btc_dom and eth_dom:
        compressed["market_summary"]["others_dominance"] = round(100 - btc_dom - eth_dom - usdt_dom, 2)
```

If CoinGecko doesn't include USDT in `market_cap_percentage`, check CMC global data:
```python
    # Fallback: try CMC for USDT dominance
    if not compressed["market_summary"]["usdt_dominance"] and isinstance(cmc_global, dict):
        # CMC might have stablecoin_volume_24h or similar
        usdt_dom_cmc = cmc_global.get("usdt_market_cap_dominance") or cmc_global.get("stablecoin_dominance")
        if usdt_dom_cmc:
            compressed["market_summary"]["usdt_dominance"] = usdt_dom_cmc
```
